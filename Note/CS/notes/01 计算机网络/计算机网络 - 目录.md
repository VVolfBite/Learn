## 使用方法

这份笔记适合学过网络但遗忘较多的情况，重点是快速唤醒记忆和理清主线，不是协议手册。

**模块学习路径**：
1. 从"网络基础总览"开始，理解分层思想
2. 通过"OSI与TCP/IP模型"建立层次框架
3. 自底向上学习各层协议（物理→链路→网络→传输→应用）
4. 重点掌握传输层（TCP三次握手、四次挥手）和应用层（HTTP/HTTPS）
5. 用"网络高频判断"查漏补缺

**时间分配建议**：
- 基础与模型（2篇）：1-2小时
- 各层协议（5篇）：3-4小时，重点在传输层
- 核心应用（3篇）：2-3小时，重点在HTTPS
- 安全与高频（2篇）：1-2小时

## 复习路径

**系统复习**（2-4 周）：顺着 TCP/IP 主线，从分层模型到应用层协议，重点在传输层和 HTTP

**快速复习**（1 周）：传输层（TCP/UDP、三次握手、四次挥手）→ HTTP/HTTPS → 高频判断

**临阵磨枪**（1 天）：高频判断 + 三次握手/四次挥手 + HTTPS 握手
