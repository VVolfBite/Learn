# 数据库进阶基础 - 目录

这组笔记不讲某一个数据库产品的操作细节，而是补关系型数据库之后最容易被忽略、却又最容易在面试和工作中被问到的通用认知。重点不是背术语，而是建立判断：什么时候该依赖数据库特性，什么时候该回到 SQL，什么时候需要扩展，什么时候要先控制复杂度。

本模块包含这些卡片：ORMs、Normalization、ACID、Failure Modes、Transactions、N+1 Problem、Database Indexes、Sharding Strategies、Data Replication、CAP Theorem、Scaling Databases、Migrations，以及最后的高频判断。它们一起回答一个更大的问题：数据库为什么既是系统最可靠的一层，也是最容易在性能、扩展和一致性上出问题的一层。

每张卡片都按“核心判断—问题背景—常见误区—面试口径—快速回忆”的节奏展开。阅读时可以按顺序建立体系，也可以把某一张卡片当作单独的复习入口。
