##  使用说明

### 笔记结构
本笔记库包含三层内容：
1. **正式主稿**：经过系统整理的正式版本，推荐优先阅读
2. **旧稿参考**：早期版本的笔记，保留作为参考资料
3. **题解辅助**：配套的题目解析（Leetcode、剑指 Offer）

### 如何使用
- **系统学习**：从各模块的目录页开始，按顺序学习
- **面试复习**：重点看高频判断和易混淆点
- **专题突破**：针对薄弱环节，重点学习某个专题
- **刷题辅助**：结合题解，理解算法思想

### 复习建议
1. 先看各模块的目录页，了解知识体系
2. 按照推荐的复习路径学习
3. 重点掌握高频考点和易混淆概念
4. 结合题解进行实战练习
