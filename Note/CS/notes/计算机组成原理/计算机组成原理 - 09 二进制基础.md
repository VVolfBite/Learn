## 二进制概述

二进制(Binary)是一种计数系统,只使用 0 和 1 两个数字。基数是 2,每一位只能是 0 或 1。从右到左,每一位的权重是 2 的幂次。

```
二进制: 1011
位权:   2³ 2² 2¹ 2⁰
       8  4  2  1
```

**对比其他进制:**

| 进制 | 基数 | 使用的数字 | 示例 | 用途 |
|------|------|-----------|------|------|
| 二进制 | 2 | 0, 1 | 1011 | CPU 内部表示 |
| 八进制 | 8 | 0-7 | 13 | 较少使用 |
| 十进制 | 10 | 0-9 | 11 | 人类习惯 |
| 十六进制 | 16 | 0-9, A-F | B | 二进制的简写 |

### 计算机使用二进制

**物理实现简单。**电路只需要两种状态:高电平(1)和低电平(0)。晶体管导通是 1,截止是 0。磁性材料正向磁化是 1,反向磁化是 0。

**抗干扰能力强。**只需要区分两种状态,不容易出错。如果使用十进制,需要区分 10 种电压,容易受干扰。

**逻辑运算简单。**布尔代数只有真(1)和假(0),与、或、非等逻辑运算可以直接用电路实现。CPU 内部的 ALU 就是用这些逻辑门电路搭建的。

**示例:**

```
二进制电路:
  高电平(3-5V) → 1
  低电平(0-0.8V) → 0
  中间电压视为干扰,容易识别

十进制电路(假设):
  0V → 0, 0.5V → 1, 1.0V → 2, ..., 4.5V → 9
  需要精确区分 10 种电压,容易出错
```

**这就是为什么 CPU 只能处理二进制数据。**所有数据(整数、小数、字符、图片)在计算机中都必须转换成二进制,CPU 才能运算。

### 二进制数的表示

**表示方法:**

```
二进制: 1011
前缀表示: 0b1011 (C/C++/Python)
后缀表示: 1011B (汇编)
下标表示: (1011)₂
```

### 进制转换

**2转10方法:** 按位权展开求和

```
(1011)₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰
        = 8 + 0 + 2 + 1
        = 11
```

**10转2方法:** 除 2 取余,逆序排列

```
11 ÷ 2 = 5 ... 余 1
 5 ÷ 2 = 2 ... 余 1
 2 ÷ 2 = 1 ... 余 0
 1 ÷ 2 = 0 ... 余 1

逆序: 1011
所以 11 = (1011)₂
```

## 位、字节、字

**位(Bit):**最小的信息单位、一个二进制数字(0 或 1)、缩写b

**字节(Byte):**8 个位组成 1 个字节，计算机中最基本的存储单位，缩写B，1 Byte = 8 bits

**字(Word):**CPU 一次处理的数据长度，32 位 CPU: 1 字 = 4 字节 = 32 位，64 位 CPU: 1 字 = 8 字节 = 64 位

**注意:** 硬盘厂商通常使用 1000 进制(1KB = 1000B),而操作系统使用 1024 进制。

## 有符号与无符号

### 无符号整数

**定义:** 只能表示非负整数(0 和正整数)。

**范围:**
- n 位无符号整数: 0 到 2ⁿ - 1
- 8 位: 0 到 255
- 16 位: 0 到 65535
- 32 位: 0 到 4,294,967,295

**示例:**

```c
unsigned char a = 255;    // 8 位,范围 0-255
unsigned int b = 4000000000;  // 32 位,范围 0-4294967295
```

**二进制表示:**

```
8 位无符号整数:
0 = 00000000
1 = 00000001
127 = 01111111
128 = 10000000
255 = 11111111
```

### 有符号整数

**定义:** 可以表示负数、零和正数。

**范围:**
- n 位有符号整数: -2ⁿ⁻¹ 到 2ⁿ⁻¹ - 1
- 8 位: -128 到 127
- 16 位: -32768 到 32767
- 32 位: -2,147,483,648 到 2,147,483,647

**表示方法:**
- 原码: 最高位为符号位,其余位表示绝对值
- 反码: 正数同原码,负数为原码除符号位外取反
- **补码**: 计算机实际使用的表示方法(详见"补码"章节)，补码的出现是为了兼容二进制级别的数值加减乘除

### 符号位

**最高位作为符号位：**0 表示正数，1 表示负数

```
01111111 = +127 (最高位 0,正数)
10000000 = -128 (最高位 1,负数)
```

**注意:** 这里使用补码表示,具体原理见"补码"章节。

## 位运算

### 基本位运算

**按位与(AND):**

```
运算符: &
规则: 两位都为 1,结果为 1

  1011
& 0110
------
  0010
```

**按位或(OR):**

```
运算符: |
规则: 至少一位为 1,结果为 1

  1011
| 0110
------
  1111
```

**按位异或(XOR):**

```
运算符: ^
规则: 两位不同,结果为 1

  1011
^ 0110
------
  1101
```

**按位取反(NOT):**

```
运算符: ~
规则: 0 变 1,1 变 0

~ 1011
------
  0100 (假设 4 位)
```

**左移:**

```
运算符: <<
规则: 向左移动,右边补 0

1011 << 1 = 10110 (乘以 2)
1011 << 2 = 101100 (乘以 4)
```

**右移:**

```
运算符: >>
规则: 向右移动,左边补 0(无符号)或补符号位(有符号)

1011 >> 1 = 0101 (除以 2)
1011 >> 2 = 0010 (除以 4)
```

### 位运算的应用

**判断奇偶:**

```c
bool isOdd(int n) {
    return n & 1;  // 最低位为 1 则为奇数
}
```

**交换两个数(不用临时变量):**

```c
void swap(int& a, int& b) {
    a = a ^ b;
    b = a ^ b;  // b = a ^ b ^ b = a
    a = a ^ b;  // a = a ^ b ^ a = b
}
```

**判断 2 的幂:**

```c
bool isPowerOfTwo(int n) {
    return n > 0 && (n & (n - 1)) == 0;
}
// 2 的幂只有一个 1: 1000, 0100, 0010, 0001
// n - 1 会将这个 1 变成 0,后面全变成 1
// n & (n - 1) 结果为 0
```

**统计 1 的个数:**

```c
int countOnes(int n) {
    int count = 0;
    while (n) {
        n = n & (n - 1);  // 每次消除最低位的 1
        count++;
    }
    return count;
}
```

**获取第 k 位:**

```c
int getBit(int n, int k) {
    return (n >> k) & 1;
}
```

**设置第 k 位为 1:**

```c
int setBit(int n, int k) {
    return n | (1 << k);
}
```

**清除第 k 位(设为 0):**

```c
int clearBit(int n, int k) {
    return n & ~(1 << k);
}
```

**快速乘除 2 的幂:**

```c
int multiplyBy8(int n) {
    return n << 3;  // 左移 3 位,相当于乘以 2³ = 8
}

int divideBy4(int n) {
    return n >> 2;  // 右移 2 位,相当于除以 2² = 4
}
```

## 高频判断

### 判断 1: 计算机使用二进制是因为物理实现简单

**正确。** 电路只需要两种状态(高低电平),容易实现且抗干扰能力强。

### 判断 2: 1 字节等于 8 位

**正确。** 1 Byte = 8 bits,这是计算机中的基本单位。

### 判断 3: 8 位无符号整数的范围是 -128 到 127

**错误。** 8 位无符号整数的范围是 0 到 255。-128 到 127 是有符号整数的范围。

### 判断 4: 二进制 1011 等于十进制 11

**正确。** 1×2³ + 0×2² + 1×2¹ + 1×2⁰ = 8 + 2 + 1 = 11。

### 判断 5: 十六进制是二进制的简写

**正确。** 1 个十六进制数字对应 4 位二进制,方便阅读和书写。

### 判断 6: 左移 1 位相当于乘以 2

**正确。** 二进制左移 1 位,相当于乘以 2。

### 判断 7: 按位与运算可以用来判断奇偶

**正确。** n & 1,如果结果为 1 则为奇数,为 0 则为偶数。

### 判断 8: n & (n-1) 可以消除最低位的 1

**正确。** 这是常用的位运算技巧,用于统计 1 的个数或判断 2 的幂。

### 判断 9: 有符号整数使用最高位作为符号位

**正确。** 最高位为 0 表示正数,为 1 表示负数。

### 判断 10: 1 KB 等于 1000 字节

**不完全正确。** 在计算机科学中,1 KB = 1024 字节(2¹⁰)。但硬盘厂商通常使用 1 KB = 1000 字节。

### 判断 11: 异或运算可以用来交换两个数

**正确。** 利用 a ^ b ^ b = a 的性质,可以不用临时变量交换两个数。

### 判断 12: 右移 1 位相当于除以 2

**正确。** 二进制右移 1 位,相当于除以 2(向下取整)。

### 判断 13: 按位取反运算会改变符号

**正确。** 对于有符号整数,按位取反会改变符号位,从而改变符号。

### 判断 14: 3 位二进制可以表示 8 个不同的数

**正确。** 2³ = 8,可以表示 0 到 7 共 8 个数。

### 判断 15: 二进制只能表示整数

**错误。** 二进制可以表示整数、浮点数、字符等各种数据,只是表示方法不同。
