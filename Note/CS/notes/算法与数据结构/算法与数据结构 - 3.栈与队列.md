# 算法与数据结构 - 栈与队列

## 1. 栈

### 1.1 栈的定义

栈（Stack）是一种后进先出（LIFO, Last In First Out）的线性数据结构。

**特点**：
- 只能在一端（栈顶）进行插入和删除操作
- 后进先出：最后压入的元素最先弹出
- 受限的线性表

### 1.2 栈的基本操作

```python
class Stack:
    def __init__(self):
        self.items = []
    
    def push(self, item):
        """入栈 - O(1)"""
        self.items.append(item)
    
    def pop(self):
        """出栈 - O(1)"""
        if not self.is_empty():
            return self.items.pop()
        return None
    
    def peek(self):
        """查看栈顶元素 - O(1)"""
        if not self.is_empty():
            return self.items[-1]
        return None
    
    def is_empty(self):
        """判断栈是否为空 - O(1)"""
        return len(self.items) == 0
    
    def size(self):
        """获取栈的大小 - O(1)"""
        return len(self.items)

# 使用示例
stack = Stack()
stack.push(1)
stack.push(2)
stack.push(3)
print(stack.pop())  # 输出: 3
print(stack.peek())  # 输出: 2
```

### 1.3 单调栈

单调栈是一种特殊的栈，栈内元素保持单调性（递增或递减）。

**应用场景**：
- 寻找下一个更大/更小的元素
- 计算直方图最大矩形面积
- 股票价格跨度问题

```python
def next_greater_element(nums):
    """
    找到每个元素右边第一个比它大的元素
    使用单调递减栈
    """
    n = len(nums)
    result = [-1] * n
    stack = []  # 存储索引
    
    for i in range(n):
        while stack and nums[stack[-1]] < nums[i]:
            idx = stack.pop()
            result[idx] = nums[i]
        stack.append(i)
    
    return result

# 示例
nums = [2, 1, 2, 4, 3]
print(next_greater_element(nums))  # 输出: [4, 2, 4, -1, -1]
```

### 1.4 栈的应用

#### 1.4.1 括号匹配

```python
def is_valid_parentheses(s):
    """
    判断括号是否匹配
    Leetcode 20. Valid Parentheses
    """
    stack = []
    mapping = {')': '(', '}': '{', ']': '['}
    
    for char in s:
        if char in mapping:
            top = stack.pop() if stack else '#'
            if mapping[char] != top:
                return False
        else:
            stack.append(char)
    
    return not stack

# 示例
print(is_valid_parentheses("()[]{}"))  # True
print(is_valid_parentheses("([)]"))    # False
```

#### 1.4.2 表达式求值

```python
def eval_rpn(tokens):
    """
    逆波兰表达式求值
    Leetcode 150. Evaluate Reverse Polish Notation
    """
    stack = []
    
    for token in tokens:
        if token in ['+', '-', '*', '/']:
            b = stack.pop()
            a = stack.pop()
            if token == '+':
                stack.append(a + b)
            elif token == '-':
                stack.append(a - b)
            elif token == '*':
                stack.append(a * b)
            else:
                stack.append(int(a / b))
        else:
            stack.append(int(token))
    
    return stack[0]

# 示例
tokens = ["2", "1", "+", "3", "*"]
print(eval_rpn(tokens))  # 输出: 9
```

## 2. 队列

### 2.1 队列的定义

队列（Queue）是一种先进先出（FIFO, First In First Out）的线性数据结构。

**特点**：
- 在队尾插入元素，在队头删除元素
- 先进先出：最先入队的元素最先出队
- 受限的线性表

### 2.2 队列的基本操作

```python
class Queue:
    def __init__(self):
        self.items = []
    
    def enqueue(self, item):
        """入队 - O(1)"""
        self.items.append(item)
    
    def dequeue(self):
        """出队 - O(N)（使用列表实现）"""
        if not self.is_empty():
            return self.items.pop(0)
        return None
    
    def front(self):
        """查看队头元素 - O(1)"""
        if not self.is_empty():
            return self.items[0]
        return None
    
    def is_empty(self):
        """判断队列是否为空 - O(1)"""
        return len(self.items) == 0
    
    def size(self):
        """获取队列大小 - O(1)"""
        return len(self.items)

# 使用 collections.deque 实现高效队列
from collections import deque

class EfficientQueue:
    def __init__(self):
        self.items = deque()
    
    def enqueue(self, item):
        """入队 - O(1)"""
        self.items.append(item)
    
    def dequeue(self):
        """出队 - O(1)"""
        if not self.is_empty():
            return self.items.popleft()
        return None
    
    def front(self):
        """查看队头元素 - O(1)"""
        if not self.is_empty():
            return self.items[0]
        return None
    
    def is_empty(self):
        return len(self.items) == 0
    
    def size(self):
        return len(self.items)
```

### 2.3 双端队列

双端队列（Deque）允许在两端进行插入和删除操作。

```python
from collections import deque

class Deque:
    def __init__(self):
        self.items = deque()
    
    def add_front(self, item):
        """在队头添加 - O(1)"""
        self.items.appendleft(item)
    
    def add_rear(self, item):
        """在队尾添加 - O(1)"""
        self.items.append(item)
    
    def remove_front(self):
        """从队头移除 - O(1)"""
        if not self.is_empty():
            return self.items.popleft()
        return None
    
    def remove_rear(self):
        """从队尾移除 - O(1)"""
        if not self.is_empty():
            return self.items.pop()
        return None
    
    def is_empty(self):
        return len(self.items) == 0
    
    def size(self):
        return len(self.items)
```

### 2.4 优先队列

优先队列中每个元素都有优先级，优先级高的元素先出队。

```python
import heapq

class PriorityQueue:
    def __init__(self):
        self.heap = []
    
    def push(self, item, priority):
        """入队 - O(log N)"""
        heapq.heappush(self.heap, (priority, item))
    
    def pop(self):
        """出队 - O(log N)"""
        if not self.is_empty():
            return heapq.heappop(self.heap)[1]
        return None
    
    def is_empty(self):
        return len(self.heap) == 0
    
    def size(self):
        return len(self.heap)

# 示例
pq = PriorityQueue()
pq.push("task1", 3)
pq.push("task2", 1)
pq.push("task3", 2)
print(pq.pop())  # 输出: task2 (优先级最低)
```

## 3. 典型题

### 3.1 用栈实现队列

**Leetcode 232. Implement Queue using Stacks**

```python
class MyQueue:
    def __init__(self):
        self.in_stack = []
        self.out_stack = []
    
    def push(self, x):
        """入队 - O(1)"""
        self.in_stack.append(x)
    
    def pop(self):
        """出队 - 均摊 O(1)"""
        self._move()
        return self.out_stack.pop()
    
    def peek(self):
        """查看队头 - 均摊 O(1)"""
        self._move()
        return self.out_stack[-1]
    
    def empty(self):
        return not self.in_stack and not self.out_stack
    
    def _move(self):
        """将 in_stack 的元素移到 out_stack"""
        if not self.out_stack:
            while self.in_stack:
                self.out_stack.append(self.in_stack.pop())
```

### 3.2 用队列实现栈

**Leetcode 225. Implement Stack using Queues**

```python
from collections import deque

class MyStack:
    def __init__(self):
        self.queue = deque()
    
    def push(self, x):
        """入栈 - O(N)"""
        self.queue.append(x)
        # 将前面的元素移到后面
        for _ in range(len(self.queue) - 1):
            self.queue.append(self.queue.popleft())
    
    def pop(self):
        """出栈 - O(1)"""
        return self.queue.popleft()
    
    def top(self):
        """查看栈顶 - O(1)"""
        return self.queue[0]
    
    def empty(self):
        return not self.queue
```

### 3.3 最小值栈

**Leetcode 155. Min Stack**

```python
class MinStack:
    def __init__(self):
        self.stack = []
        self.min_stack = []
    
    def push(self, val):
        """入栈 - O(1)"""
        self.stack.append(val)
        if not self.min_stack or val <= self.min_stack[-1]:
            self.min_stack.append(val)
    
    def pop(self):
        """出栈 - O(1)"""
        if self.stack:
            val = self.stack.pop()
            if val == self.min_stack[-1]:
                self.min_stack.pop()
    
    def top(self):
        """查看栈顶 - O(1)"""
        return self.stack[-1] if self.stack else None
    
    def get_min(self):
        """获取最小值 - O(1)"""
        return self.min_stack[-1] if self.min_stack else None
```

### 3.4 滑动窗口最大值

**Leetcode 239. Sliding Window Maximum**

```python
from collections import deque

def max_sliding_window(nums, k):
    """
    使用单调递减双端队列
    时间复杂度 O(N)
    """
    if not nums or k == 0:
        return []
    
    dq = deque()  # 存储索引
    result = []
    
    for i in range(len(nums)):
        # 移除窗口外的元素
        while dq and dq[0] < i - k + 1:
            dq.popleft()
        
        # 维护单调递减队列
        while dq and nums[dq[-1]] < nums[i]:
            dq.pop()
        
        dq.append(i)
        
        # 窗口形成后记录最大值
        if i >= k - 1:
            result.append(nums[dq[0]])
    
    return result

# 示例
nums = [1, 3, -1, -3, 5, 3, 6, 7]
k = 3
print(max_sliding_window(nums, k))  # 输出: [3, 3, 5, 5, 6, 7]
```

### 3.5 每日温度

**Leetcode 739. Daily Temperatures**

```python
def daily_temperatures(temperatures):
    """
    使用单调栈
    时间复杂度 O(N)
    """
    n = len(temperatures)
    result = [0] * n
    stack = []  # 存储索引
    
    for i in range(n):
        while stack and temperatures[stack[-1]] < temperatures[i]:
            idx = stack.pop()
            result[idx] = i - idx
        stack.append(i)
    
    return result

# 示例
temperatures = [73, 74, 75, 71, 69, 72, 76, 73]
print(daily_temperatures(temperatures))  # 输出: [1, 1, 4, 2, 1, 1, 0, 0]
```

## 4. 高频判断

### 4.1 栈与队列的选择

**Q1：什么时候用栈，什么时候用队列？**

A：
- 需要后进先出 → 栈（函数调用、括号匹配、表达式求值）
- 需要先进先出 → 队列（BFS、任务调度、缓冲区）
- 需要找最近的更大/更小元素 → 单调栈
- 需要滑动窗口最值 → 单调队列

**Q2：如何用两个栈实现队列？**

A：
- 一个栈用于入队（in_stack）
- 一个栈用于出队（out_stack）
- 出队时如果 out_stack 为空，将 in_stack 的元素全部移到 out_stack
- 均摊时间复杂度 O(1)

**Q3：如何实现一个支持 O(1) 获取最小值的栈？**

A：
- 使用辅助栈记录每个状态的最小值
- 入栈时，如果新元素 ≤ 当前最小值，同时入辅助栈
- 出栈时，如果出栈元素 = 当前最小值，辅助栈也出栈

**Q4：单调栈的应用场景？**

A：
- 下一个更大元素
- 下一个更小元素
- 直方图最大矩形
- 接雨水问题

**Q5：优先队列的底层实现？**

A：
- 通常使用堆（二叉堆）实现
- 入队：O(log N)
- 出队：O(log N)
- 查看队头：O(1)
