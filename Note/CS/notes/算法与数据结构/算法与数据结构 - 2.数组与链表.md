
# 算法与数据结构 - 数组与链表

## 1. 数组

### 1.1 数组的定义

数组是一种线性数据结构，用连续的内存空间存储相同类型的元素集合。

**特点**：
- 连续存储：元素在内存中连续排列
- 随机访问：通过索引可以 O(1) 时间访问任意元素
- 固定大小：大多数语言中数组大小在创建时确定

### 1.2 数组的存储结构

```
内存地址：  1000   1004   1008   1012   1016
数组元素：  [10]   [20]   [30]   [40]   [50]
索引：       0      1      2      3      4
```

**地址计算公式**：
```
元素地址 = 起始地址 + 索引 × 元素大小
```

### 1.3 数组的基本操作

#### 1.3.1 访问元素

**时间复杂度**：O(1)

```python
def access(arr, index):
    """访问数组元素"""
    if 0 <= index < len(arr):
        return arr[index]
    return None

# 示例
arr = [10, 20, 30, 40, 50]
print(access(arr, 2))  # 输出: 30
```

#### 1.3.2 插入元素

**时间复杂度**：
- 尾部插入：O(1)
- 头部插入：O(N)
- 中间插入：O(N)

```python
def insert(arr, index, value):
    """在指定位置插入元素"""
    # Python 列表的 insert 方法
    arr.insert(index, value)
    return arr

# 手动实现（需要移动元素）
def insert_manual(arr, index, value):
    """手动实现插入（模拟固定大小数组）"""
    n = len(arr)
    # 创建新数组
    new_arr = [0] * (n + 1)
    
    # 复制 index 之前的元素
    for i in range(index):
        new_arr[i] = arr[i]
    
    # 插入新元素
    new_arr[index] = value
    
    # 复制 index 之后的元素
    for i in range(index, n):
        new_arr[i + 1] = arr[i]
    
    return new_arr

# 示例
arr = [10, 20, 30, 40, 50]
arr = insert(arr, 2, 25)
print(arr)  # 输出: [10, 20, 25, 30, 40, 50]
```

#### 1.3.3 删除元素

**时间复杂度**：
- 尾部删除：O(1)
- 头部删除：O(N)
- 中间删除：O(N)

```python
def delete(arr, index):
    """删除指定位置的元素"""
    if 0 <= index < len(arr):
        arr.pop(index)
    return arr

# 手动实现
def delete_manual(arr, index):
    """手动实现删除"""
    n = len(arr)
    if index < 0 or index >= n:
        return arr
    
    # 创建新数组
    new_arr = [0] * (n - 1)
    
    # 复制 index 之前的元素
    for i in range(index):
        new_arr[i] = arr[i]
    
    # 复制 index 之后的元素
    for i in range(index + 1, n):
        new_arr[i - 1] = arr[i]
    
    return new_arr

# 示例

arr = delete(arr, 2)
print(arr)  # 输出: [10, 20, 40, 50]
```

#### 1.3.4 查找元素

**时间复杂度**：
- 无序数组：O(N)
- 有序数组（二分查找）：O(log N)

```python
def search(arr, target):
    """在数组中查找元素"""
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

# 二分查找（有序数组）
def binary_search(arr, target):
    """二分查找"""
    left, right = 0, len(arr) - 1
    
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return -1

# 示例
arr = [10, 20, 30, 40, 50]
print(search(arr, 30))  # 输出: 2
print(binary_search(arr, 30))  # 输出: 2
```

## 2. 链表

### 2.1 链表的定义

链表是一种线性数据结构，通过指针将一系列节点连接起来，每个节点包含数据和指向下一个节点的指针。

**特点**：
- 非连续存储：节点在内存中可以不连续
- 动态大小：可以方便地增加或删除节点
- 顺序访问：需要从头节点开始遍历

### 2.2 链表的类型

#### 2.2.1 单链表

每个节点包含数据和指向下一个节点的指针。

```
[10|next] -> [20|next] -> [30|next] -> [40|null]
  head                                    tail
```

**节点定义**：

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next
```

#### 2.2.2 双向链表

每个节点包含数据、指向前一个节点的指针和指向下一个节点的指针。

```
null <- [prev|10|next] <-> [prev|20|next] <-> [prev|30|null]
          head                                    tail
```

**节点定义**：

```python
class DoublyListNode:
    def __init__(self, val=0, prev=None, next=None):
        self.val = val
        self.prev = prev
        self.next = next
```

#### 2.2.3 循环链表

尾节点的 next 指针指向头节点，形成一个环。

```
[10|next] -> [20|next] -> [30|next] -> [40|next]
  ^                                        |
  |________________________________________|
```

### 2.3 单链表的基本操作

#### 2.3.1 创建链表

```python
def create_linked_list(arr):
    """从数组创建链表"""
    if not arr:
        return None
    
    head = ListNode(arr[0])
    current = head
    
    for i in range(1, len(arr)):
        current.next = ListNode(arr[i])
        current = current.next
    
    return head

# 示例
arr = [10, 20, 30, 40, 50]
head = create_linked_list(arr)
```

#### 2.3.2 插入节点

**时间复杂度**：
- 头部插入：O(1)
- 尾部插入：O(N)（需要遍历到尾部）
- 中间插入：O(N)

```python
def insert_at_head(head, val):
    """在头部插入节点"""
    new_node = ListNode(val)
    new_node.next = head
    return new_node

def insert_at_tail(head, val):
    """在尾部插入节点"""
    new_node = ListNode(val)
    
    if not head:
        return new_node
    
    current = head
    while current.next:
        current = current.next
    
    current.next = new_node
    return head

def insert_at_position(head, val, position):
    """在指定位置插入节点"""
    if position == 0:
        return insert_at_head(head, val)
    
    new_node = ListNode(val)
    current = head
    
    for i in range(position - 1):
        if not current:
            return head
        current = current.next
    
    if current:
        new_node.next = current.next
        current.next = new_node
    
    return head

# 示例
head = create_linked_list([10, 20, 30])
head = insert_at_head(head, 5)
head = insert_at_tail(head, 40)
head = insert_at_position(head, 25, 3)
```

#### 2.3.3 删除节点

**时间复杂度**：
- 头部删除：O(1)
- 尾部删除：O(N)
- 中间删除：O(N)

```python
def delete_at_head(head):
    """删除头节点"""
    if not head:
        return None
    return head.next

def delete_at_tail(head):
    """删除尾节点"""
    if not head or not head.next:
        return None
    
    current = head
    while current.next.next:
        current = current.next
    
    current.next = None
    return head

def delete_by_value(head, val):
    """删除指定值的节点"""
    if not head:
        return None
    
    # 如果头节点就是要删除的节点
    if head.val == val:
        return head.next
    
    current = head
    while current.next:
        if current.next.val == val:
            current.next = current.next.next
            return head
        current = current.next
    
    return head

def delete_at_position(head, position):
    """删除指定位置的节点"""
    if not head:
        return None
    
    if position == 0:
        return head.next
    
    current = head
    for i in range(position - 1):
        if not current or not current.next:
            return head
        current = current.next
    
    if current.next:
        current.next = current.next.next
    
    return head

# 示例
head = create_linked_list([10, 20, 30, 40, 50])
head = delete_at_head(head)
head = delete_at_tail(head)
head = delete_by_value(head, 30)
```

### 2.4 双向链表的基本操作

```python
class DoublyLinkedList:
    def __init__(self):
        self.head = None
        self.tail = None
    
    def insert_at_head(self, val):
        """在头部插入"""
        new_node = DoublyListNode(val)
        
        if not self.head:
            self.head = self.tail = new_node
        else:
            new_node.next = self.head
            self.head.prev = new_node
            self.head = new_node
    
    def insert_at_tail(self, val):
        """在尾部插入"""
        new_node = DoublyListNode(val)
        
        if not self.tail:
            self.head = self.tail = new_node
        else:
            new_node.prev = self.tail
            self.tail.next = new_node
            self.tail = new_node
    
    def delete_at_head(self):
        """删除头节点"""
        if not self.head:
            return
        
        if self.head == self.tail:
            self.head = self.tail = None
        else:
            self.head = self.head.next
            self.head.prev = None
    
    def delete_at_tail(self):
        """删除尾节点"""
        if not self.tail:
            return
        
        if self.head == self.tail:
            self.head = self.tail = None
        else:
            self.tail = self.tail.prev
            self.tail.next = None
```

```python
def is_circular(head):
    """判断是否为循环链表"""
    if not head:
        return False
    
    slow = fast = head
    
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        
        if slow == fast:
            return True
    
    return False

def create_circular_list(arr):

        return None

    current = head
    
    for i in range(1, len(arr)):
        current.next = ListNode(arr[i])
        current = current.next
    
    # 尾节点指向头节点
    current.next = head
    
    return head
```

## 3. 顺序存储与链式存储

### 3.1 对比

| 特性 | 数组（顺序存储） | 链表（链式存储） |
| :---: | :---: | :---: |
| 存储方式 | 连续内存 | 非连续内存 |
| 访问元素 | O(1) | O(N) |
| 插入元素 | O(N) | O(1)（已知位置） |
| 删除元素 | O(N) | O(1)（已知位置） |
| 空间利用 | 可能浪费 | 按需分配 |
| 缓存性能 | 好 | 差 |
| 实现复杂度 | 简单 | 相对复杂 |

**使用数组的场景**：
- 需要频繁随机访问元素
- 数据规模固定或变化不大
- 对缓存性能要求高
- 实现简单的数据结构（栈、队列）

**使用链表的场景**：
- 需要频繁插入删除元素
- 数据规模动态变化
- 不需要随机访问
- 实现复杂的数据结构（图、树）

```python
def two_sum(nums, target):
    """
    使用哈希表，时间复杂度 O(N)，空间复杂度 O(N)
    """
    hash_map = {}
    
    for i, num in enumerate(nums):
        complement = target - num
        if complement in hash_map:
            return [hash_map[complement], i]
        hash_map[num] = i
    
    return []

# 示例
```

```python
def find_duplicate(nums):
    """
    在 0~n-1 范围内的数组中找重复数字
    """
    for i in range(len(nums)):
        while nums[i] != i:
            if nums[i] == nums[nums[i]]:
                return nums[i]
            # 交换
            nums[nums[i]], nums[i] = nums[i], nums[nums[i]]
    
    return -1

# 示例
```

```python
def search_matrix(matrix, target):
    """
    从右上角开始查找
    时间复杂度 O(M+N)
    """
    if not matrix or not matrix[0]:
        return False
    
    rows, cols = len(matrix), len(matrix[0])
    row, col = 0, cols - 1
    
    while row < rows and col >= 0:
        if matrix[row][col] == target:
            return True
        elif matrix[row][col] > target:
            col -= 1
        else:
            row += 1
    
    return False

# 示例

    [2,  5,  8,  12, 19],
    [3,  6,  9,  16, 22],
    [10, 13, 14, 17, 24],
    [18, 21, 23, 26, 30]
]
print(search_matrix(matrix, 5))  # 输出: True
```

```python
def reverse_list(head):
    """
    迭代法反转链表
    时间复杂度 O(N)，空间复杂度 O(1)
    """
    prev = None
    current = head
    
    while current:
        next_node = current.next
        current.next = prev
        prev = current
        current = next_node
    
    return prev

# 递归法
def reverse_list_recursive(head):
    """
    递归法反转链表
    """
    if not head or not head.next:
        return head
    
    new_head = reverse_list_recursive(head.next)
    head.next.next = head
    head.next = None
    
    return new_head
```

#### 4.2.2 合并两个有序链表

**Leetcode 21. Merge Two Sorted Lists (Easy)**

```python
def merge_two_lists(l1, l2):
    """
    合并两个有序链表
    """
    dummy = ListNode(0)
    current = dummy
    
    while l1 and l2:
        if l1.val < l2.val:
            current.next = l1
            l1 = l1.next
        else:
            current.next = l2
            l2 = l2.next
        current = current.next
    
    current.next = l1 if l1 else l2
    
    return dummy.next

# 递归法
def merge_two_lists_recursive(l1, l2):
    """
    递归法合并
    """
    if not l1:
        return l2
    if not l2:
        return l1
    
    if l1.val < l2.val:
        l1.next = merge_two_lists_recursive(l1.next, l2)
        return l1
    else:
        l2.next = merge_two_lists_recursive(l1, l2.next)
        return l2
```

#### 4.2.3 链表中环的检测

**Leetcode 141. Linked List Cycle (Easy)**

```python
def has_cycle(head):
    """
    使用快慢指针检测环
    时间复杂度 O(N)，空间复杂度 O(1)
    """
    if not head or not head.next:
        return False
    
    slow = head
    fast = head.next
    
    while slow != fast:
        if not fast or not fast.next:
            return False
        slow = slow.next
        fast = fast.next.next
    
    return True

# 找到环的入口
def detect_cycle(head):
    """
    找到环的入口节点
    """
    if not head or not head.next:
        return None
    
    slow = fast = head
    
    # 判断是否有环
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        
        if slow == fast:
            # 有环，找入口
            slow = head
            while slow != fast:
                slow = slow.next
                fast = fast.next
            return slow
    
    return None
```

#### 4.2.4 删除链表的倒数第 N 个节点

**Leetcode 19. Remove Nth Node From End of List (Medium)**

```python
def remove_nth_from_end(head, n):
    """
    使用双指针
    时间复杂度 O(N)，空间复杂度 O(1)
    """
    dummy = ListNode(0)
    dummy.next = head
    
    fast = slow = dummy
    
    # fast 先走 n+1 步
    for i in range(n + 1):
        fast = fast.next
    
    # fast 和 slow 一起走
    while fast:
        fast = fast.next
        slow = slow.next
    
    # 删除节点
    slow.next = slow.next.next
    
    return dummy.next
```

### 5.1 数组 vs 链表选择

**Q1：什么时候用数组，什么时候用链表？**

A：
- 需要频繁随机访问 → 数组
- 需要频繁插入删除 → 链表
- 数据规模固定 → 数组
- 数据规模动态变化 → 链表

**Q2：为什么数组访问快？**

A：
- 连续内存存储，通过索引直接计算地址
- 利用 CPU 缓存，访问相邻元素更快
- 地址计算公式：地址 = 起始地址 + 索引 × 元素大小

**Q3：链表的插入删除真的是 O(1) 吗？**

A：
- 如果已知要操作的节点位置，插入删除是 O(1)
- 但通常需要先查找节点，查找是 O(N)
- 所以实际上链表的插入删除通常是 O(N)

**Q4：如何判断链表是否有环？**

A：
- 使用快慢指针（Floyd 判圈算法）
- 慢指针每次走一步，快指针每次走两步
- 如果有环，快慢指针一定会相遇

**Q5：如何找到链表的中间节点？**

A：
- 使用快慢指针
- 慢指针每次走一步，快指针每次走两步
- 当快指针到达末尾时，慢指针在中间

### 5.2 常见面试问题

**Q1：如何在 O(1) 时间删除链表节点？**

A：如果允许修改节点值，可以将下一个节点的值复制到当前节点，然后删除下一个节点。

```python
def delete_node(node):
    """O(1) 删除节点（不是尾节点）"""
    node.val = node.next.val
    node.next = node.next.next
```

**Q2：如何判断两个链表是否相交？**

A：
- 方法1：使用哈希表记录一个链表的所有节点
- 方法2：将一个链表的尾部连接到另一个链表的头部，判断是否有环

**Q3：数组和链表的内存占用对比？**

A：
- 数组：只存储数据，内存连续
- 链表：存储数据 + 指针，内存不连续
- 对于 n 个整数（4字节）：
  - 数组：4n 字节
  - 单链表：8n 字节（数据4字节 + 指针4字节）
  - 双向链表：12n 字节（数据4字节 + 两个指针8字节）
