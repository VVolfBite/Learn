## 基本位运算

**与（AND）**：`a & b`，两位都为 1 时结果为 1
```
  1010
& 1100
------
  1000
```

**或（OR）**：`a | b`，至少有一位为 1 时结果为 1
```
  1010
| 1100
------
  1110
```

**异或（XOR）**：`a ^ b`，两位不同时结果为 1
```
  1010
^ 1100
------
  0110
```

**取反（NOT）**：`~a`，0 变 1，1 变 0
```
~ 1010
------
  0101
```

**左移**：`a << n`，左移 n 位，相当于乘以 2^n
```
1010 << 2 = 101000
```

**右移**：`a >> n`，右移 n 位，相当于除以 2^n
```
1010 >> 2 = 10
```

## 常用技巧

### 判断奇偶

```python
# 判断是否为奇数
if n & 1:
    print("奇数")
```

最低位为 1 表示奇数，为 0 表示偶数。

### 交换两个数

```python
a = a ^ b
b = a ^ b
a = a ^ b
```

利用异或的性质：`a ^ a = 0`，`a ^ 0 = a`。

### 判断是否为 2 的幂

```python
def is_power_of_two(n):
    return n > 0 and (n & (n - 1)) == 0
```

2 的幂的二进制表示只有一个 1，例如 8 = 1000。`n - 1` 会将最低位的 1 变为 0，其后的位全变为 1。两者相与结果为 0。

### 获取最低位的 1

```python
lowest_bit = n & (-n)
```

`-n` 是 `n` 的补码，等于 `~n + 1`。`n & (-n)` 会保留最低位的 1，其他位清零。

例如：
```
n     = 1010100
-n    = 0101100
n&(-n)= 0000100
```

### 清除最低位的 1

```python
n = n & (n - 1)
```

这个操作会将最低位的 1 变为 0。可以用来统计 1 的个数。

### 统计 1 的个数

```python
def count_ones(n):
    count = 0
    while n:
        n = n & (n - 1)
        count += 1
    return count
```

每次清除最低位的 1，直到 n 变为 0。

### 获取第 k 位

```python
# 获取第 k 位（从右往左，从 0 开始）
bit = (n >> k) & 1
```

### 设置第 k 位为 1

```python
n = n | (1 << k)
```

### 清除第 k 位（设为 0）

```python
n = n & ~(1 << k)
```

### 翻转第 k 位

```python
n = n ^ (1 << k)
```

## 异或的性质

这里真正要说明的是“异或的性质”为什么值得单独拿出来讲。把它放到“算法与数据结构 - 13.位运算”这张卡片的上下文里看，后面的要点和区别会更容易读顺。

异或有一些特殊性质，在算法中很有用：

1. **交换律**：`a ^ b = b ^ a`
2. **结合律**：`(a ^ b) ^ c = a ^ (b ^ c)`
3. **自反性**：`a ^ a = 0`
4. **恒等性**：`a ^ 0 = a`

利用这些性质可以解决一些问题。

### 找出数组中唯一的单独元素

数组中除了一个元素外，其他元素都出现两次。找出这个单独的元素。

```python
def single_number(nums):
    result = 0
    for num in nums:
        result ^= num
    return result
```

因为 `a ^ a = 0`，所以成对的元素会相互抵消，最后剩下的就是单独的元素。

### 找出两个单独的元素

数组中有两个元素只出现一次，其他元素都出现两次。找出这两个元素。

```python
def single_number_two(nums):
    # 先得到两个单独元素的异或结果
    xor = 0
    for num in nums:
        xor ^= num
    
    # 找到 xor 中任意一个为 1 的位
    # 这一位在两个单独元素中一个为 0，一个为 1
    diff_bit = xor & (-xor)
    
    # 根据这一位将数组分为两组
    a = b = 0
    for num in nums:
        if num & diff_bit:
            a ^= num
        else:
            b ^= num
    
    return [a, b]
```

## 位掩码

位掩码用一个整数的每一位表示某个元素的状态（存在或不存在）。常用于集合操作和状态压缩。

### 子集枚举

枚举集合 {0, 1, 2, ..., n-1} 的所有子集：

```python
def subsets(n):
    result = []
    for mask in range(1 << n):  # 2^n 个子集
        subset = []
        for i in range(n):
            if mask & (1 << i):
                subset.append(i)
        result.append(subset)
    return result
```

### 判断子集关系

```python
# 判断 a 是否是 b 的子集
def is_subset(a, b):
    return (a & b) == a
```

### 集合的并、交、差

```python
# 并集
union = a | b

# 交集
intersection = a & b

# 差集（a 中有但 b 中没有的元素）
difference = a & ~b
```

## 实际应用

### 权限管理

用位掩码表示用户权限：

```python
READ = 1 << 0    # 0001
WRITE = 1 << 1   # 0010
EXECUTE = 1 << 2 # 0100
DELETE = 1 << 3  # 1000

# 赋予读写权限
permission = READ | WRITE

# 检查是否有写权限
if permission & WRITE:
    print("有写权限")

# 撤销写权限
permission = permission & ~WRITE

# 切换执行权限
permission = permission ^ EXECUTE
```

### 状态压缩 DP

在动态规划中，如果状态数量较少（通常不超过 20），可以用整数的二进制位表示状态，节省空间。

例如旅行商问题（TSP），用位掩码表示已访问的城市集合。

## 常见面试问题

**Q1：为什么位运算快？**

位运算直接操作二进制位，是 CPU 的基本指令，不需要额外的计算。相比之下，乘除法需要多个时钟周期。

**Q2：`n & (n-1)` 的作用？**

清除最低位的 1。可以用来判断是否为 2 的幂，或统计 1 的个数。

**Q3：如何不用临时变量交换两个数？**

用异或：`a ^= b; b ^= a; a ^= b;`。但实际编程中不推荐，因为可读性差，且现代编译器能优化普通的交换。

**Q4：位运算的应用场景？**

- 权限管理
- 状态压缩
- 集合操作
- 优化性能（判断奇偶、乘除 2）
- 特定算法（找单独元素）
