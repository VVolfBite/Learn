## 1. 复杂度分析

### 时间复杂度
- **O(1)**: 常数时间 - 数组访问
- **O(log N)**: 对数时间 - 二分查找
- **O(N)**: 线性时间 - 遍历数组
- **O(N log N)**: 线性对数 - 快速排序、归并排序
- **O(N²)**: 平方时间 - 冒泡排序
- **O(2ᴺ)**: 指数时间 - 递归斐波那契

### Big O / Omega / Theta
- **Big O**: 上界（最坏情况）
- **Big Omega**: 下界（最好情况）
- **Big Theta**: 确界（平均情况）

### 空间复杂度
- 递归深度
- 辅助数据结构
- 原地算法 vs 非原地算法

## 2. 数组与链表

### 数组
- **访问**: O(1)
- **插入/删除**: O(N)
- **优点**: 随机访问快，缓存友好
- **缺点**: 固定大小，插入删除慢

### 链表
- **访问**: O(N)
- **插入/删除**: O(1)（已知位置）
- **类型**: 单链表、双向链表、循环链表
- **优点**: 动态大小，插入删除快
- **缺点**: 随机访问慢，额外空间

### 典型题
- 反转链表
- 合并两个有序链表
- 链表环检测
- 删除倒数第N个节点

## 8. 查找算法

### 顺序查找
- **时间复杂度**: O(N)
- **适用**: 无序数组

### 二分查找
- **时间复杂度**: O(log N)
- **前提**: 有序数组
- **边界问题**: 左闭右闭、左闭右开

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

### 典型题
- 搜索插入位置
- 搜索旋转排序数组
- 寻找峰值
- 在排序数组中查找元素的第一个和最后一个位置

## 9. 排序算法

### 比较排序

| 算法 | 时间复杂度（平均） | 空间复杂度 | 稳定性 |
|------|-------------------|-----------|--------|
| 冒泡排序 | O(N²) | O(1) | 稳定 |
| 选择排序 | O(N²) | O(1) | 不稳定 |
| 插入排序 | O(N²) | O(1) | 稳定 |
| 希尔排序 | O(N^1.3) | O(1) | 不稳定 |
| 快速排序 | O(N log N) | O(log N) | 不稳定 |
| 归并排序 | O(N log N) | O(N) | 稳定 |
| 堆排序 | O(N log N) | O(1) | 不稳定 |

### 非比较排序

| 算法 | 时间复杂度 | 空间复杂度 | 适用场景 |
|------|-----------|-----------|---------|
| 计数排序 | O(N+K) | O(K) | 整数，范围小 |
| 桶排序 | O(N+K) | O(N+K) | 均匀分布 |
| 基数排序 | O(N×K) | O(N+K) | 整数，位数少 |

### 快速排序实现

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)
```

## 11. 动态规划

### 基本思想
- **状态定义**: dp[i] 表示什么
- **状态转移**: dp[i] 如何从 dp[i-1] 推导
- **初始状态**: dp[0] 的值
- **返回结果**: dp[n] 或其他

### 经典问题

#### 线性DP
- 爬楼梯: dp[i] = dp[i-1] + dp[i-2]
- 打家劫舍: dp[i] = max(dp[i-1], dp[i-2] + nums[i])

#### 背包问题
- **0-1背包**: 每个物品只能选一次
- **完全背包**: 每个物品可以选无限次
- **多重背包**: 每个物品有数量限制

```python
# 0-1背包
def knapsack(weights, values, capacity):
    n = len(weights)
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]
    
    for i in range(1, n + 1):
        for w in range(capacity + 1):
            if weights[i-1] <= w:
                dp[i][w] = max(dp[i-1][w], 
                              dp[i-1][w-weights[i-1]] + values[i-1])
            else:
                dp[i][w] = dp[i-1][w]
    
    return dp[n][capacity]
```

#### 最长公共子序列

```python
def lcs(text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i-1] == text2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    
    return dp[m][n]
```

## 12. 贪心算法

### 基本思想
- 每一步都做出当前看起来最优的选择
- 局部最优 → 全局最优

### 适用条件
- **贪心选择性质**: 局部最优能导致全局最优
- **最优子结构**: 问题的最优解包含子问题的最优解

### 典型题
- 分发饼干
- 跳跃游戏
- 买卖股票的最佳时机II
- 无重叠区间

```python
# 跳跃游戏
def can_jump(nums):
    max_reach = 0
    for i in range(len(nums)):
        if i > max_reach:
            return False
        max_reach = max(max_reach, i + nums[i])
    return True
```

## 13. 位运算

### 基本操作
- **与(&)**: 都为1才为1
- **或(|)**: 有1就为1
- **异或(^)**: 不同为1
- **非(~)**: 取反

## 14. 字符串与模式匹配

### KMP算法
- **时间复杂度**: O(M+N)
- **核心**: next数组（部分匹配表）

```python
def kmp_search(text, pattern):
    def get_next(pattern):
        next_arr = [0] * len(pattern)
        j = 0
        for i in range(1, len(pattern)):
            while j > 0 and pattern[i] != pattern[j]:
                j = next_arr[j-1]
            if pattern[i] == pattern[j]:
                j += 1
            next_arr[i] = j
        return next_arr
    
    next_arr = get_next(pattern)
    j = 0
    for i in range(len(text)):
        while j > 0 and text[i] != pattern[j]:
            j = next_arr[j-1]
        if text[i] == pattern[j]:
            j += 1
        if j == len(pattern):
            return i - j + 1
    return -1
```

- **应用**: 前缀匹配、自动补全

## 15. 双指针与滑动窗口

### 双指针
- **快慢指针**: 链表环检测、找中点
- **左右指针**: 两数之和、反转数组

### 滑动窗口
- **固定窗口**: 固定大小的子数组
- **可变窗口**: 满足条件的最小/最大子数组

```python
# 最长无重复子串
def length_of_longest_substring(s):
    char_set = set()
    left = 0
    max_len = 0
    
    for right in range(len(s)):
        while s[right] in char_set:
            char_set.remove(s[left])
            left += 1
        char_set.add(s[right])
        max_len = max(max_len, right - left + 1)
    
    return max_len
```

## 16. 并查集与Trie

### 并查集
- **操作**: find, union - 接近 O(1)
- **应用**: 连通性问题、最小生成树

```python
class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n
    
    def find(self, x):
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]
    
    def union(self, x, y):
        px, py = self.find(x), self.find(y)
        if px == py:
            return False
        if self.rank[px] < self.rank[py]:
            px, py = py, px
        self.parent[py] = px
        if self.rank[px] == self.rank[py]:
            self.rank[px] += 1
        return True
```

## 17. 算法专题高频判断

### 数据结构选型
- **需要快速查找** → 哈希表
- **需要有序** → 平衡树
- **需要最值** → 堆
- **需要LIFO** → 栈
- **需要FIFO** → 队列

### 算法选型
- **无序查找** → 哈希表 O(1)
- **有序查找** → 二分查找 O(log N)
- **最短路径** → BFS/Dijkstra
- **连通性** → DFS/并查集
- **最优解** → 动态规划/贪心

### 复杂度优化
- **O(N²) → O(N log N)**: 排序 + 双指针
- **O(N²) → O(N)**: 哈希表
- **O(2ᴺ) → O(N²)**: 动态规划

### 常见题型入口
- **数组**: 双指针、滑动窗口、前缀和
- **链表**: 快慢指针、反转、合并
- **树**: 递归、遍历、BST性质
- **图**: BFS、DFS、最短路径
- **字符串**: 双指针、KMP、动态规划
