## 1. 哈希表基础

### 1.1 定义

哈希表（Hash Table）是一种通过哈希函数将键映射到存储位置的数据结构，实现快速的插入、删除和查找操作。

**特点**：
- 平均时间复杂度：O(1) 查找、插入、删除
- 空间复杂度：O(N)
- 无序存储

### 1.2 基本概念

这里真正要说明的是“1.2 基本概念”为什么值得单独拿出来讲。把它放到“算法与数据结构 - 4.哈希表”这张卡片的上下文里看，后面的要点和区别会更容易读顺。

- **键（Key）**：用于查找的标识
- **值（Value）**：存储的数据
- **哈希函数（Hash Function）**：将键映射到索引的函数
- **哈希冲突（Hash Collision）**：不同的键映射到相同的索引

## 2. 哈希函数

### 2.1 常见哈希函数

```python
# 除留余数法
def hash_mod(key, size):
    return key % size

# 乘法哈希
def hash_multiply(key, size):
    A = 0.6180339887  # (√5 - 1) / 2
    return int(size * ((key * A) % 1))

# 字符串哈希
def hash_string(s, size):
    hash_val = 0
    for char in s:
        hash_val = (hash_val * 31 + ord(char)) % size
    return hash_val
```

### 2.2 好的哈希函数特点

- 计算简单快速
- 分布均匀，减少冲突
- 确定性：相同输入产生相同输出

## 3. 哈希冲突解决

### 3.1 链地址法（Chaining）

将冲突的元素存储在链表中。

```python
class HashTableChaining:
    def __init__(self, size=10):
        self.size = size
        self.table = [[] for _ in range(size)]
    
    def _hash(self, key):
        return hash(key) % self.size
    
    def put(self, key, value):
        """插入 - 平均 O(1)"""
        index = self._hash(key)
        # 检查是否已存在
        for i, (k, v) in enumerate(self.table[index]):
            if k == key:
                self.table[index][i] = (key, value)
                return
        # 不存在则添加
        self.table[index].append((key, value))
    
    def get(self, key):
        """查找 - 平均 O(1)"""
        index = self._hash(key)
        for k, v in self.table[index]:
            if k == key:
                return v
        return None
    
    def remove(self, key):
        """删除 - 平均 O(1)"""
        index = self._hash(key)
        for i, (k, v) in enumerate(self.table[index]):
            if k == key:
                del self.table[index][i]
                return True
        return False

# 示例
ht = HashTableChaining()
ht.put("apple", 5)
ht.put("banana", 3)
print(ht.get("apple"))  # 输出: 5
```

### 3.2 开放寻址法（Open Addressing）

发生冲突时，寻找下一个空位置。

#### 线性探测

```python
class HashTableLinearProbing:
    def __init__(self, size=10):
        self.size = size
        self.keys = [None] * size
        self.values = [None] * size
    
    def _hash(self, key):
        return hash(key) % self.size
    
    def put(self, key, value):
        """插入 - 平均 O(1)"""
        index = self._hash(key)
        
        while self.keys[index] is not None:
            if self.keys[index] == key:
                self.values[index] = value
                return
            index = (index + 1) % self.size
        
        self.keys[index] = key
        self.values[index] = value
    
    def get(self, key):
        """查找 - 平均 O(1)"""
        index = self._hash(key)
        
        while self.keys[index] is not None:
            if self.keys[index] == key:
                return self.values[index]
            index = (index + 1) % self.size
        
        return None
```

## 4. 哈希表的操作

### 4.1 增删改查

```python
# Python 内置字典
hash_map = {}

# 插入/更新
hash_map["key1"] = "value1"
hash_map["key2"] = "value2"

# 查找
value = hash_map.get("key1")  # 返回 "value1"
value = hash_map.get("key3", "default")  # 返回 "default"

# 删除
del hash_map["key1"]
# 或
hash_map.pop("key2", None)

# 判断键是否存在
if "key1" in hash_map:
    print("存在")

# 遍历
for key, value in hash_map.items():
    print(f"{key}: {value}")
```

### 4.2 Set 操作

```python
# Python 内置集合
hash_set = set()

# 添加
hash_set.add(1)
hash_set.add(2)

# 删除
hash_set.remove(1)  # 不存在会报错
hash_set.discard(3)  # 不存在不报错

# 判断存在
if 2 in hash_set:
    print("存在")

# 集合运算
set1 = {1, 2, 3}
set2 = {2, 3, 4}
print(set1 & set2)  # 交集: {2, 3}
print(set1 | set2)  # 并集: {1, 2, 3, 4}
print(set1 - set2)  # 差集: {1}
```

## 5. 典型题

### 5.1 两数之和

**Leetcode 1. Two Sum**

```python
def two_sum(nums, target):
    """使用哈希表 - O(N)"""
    hash_map = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in hash_map:
            return [hash_map[complement], i]
        hash_map[num] = i
    return []
```

### 5.2 字母异位词分组

**Leetcode 49. Group Anagrams**

```python
def group_anagrams(strs):
    """使用哈希表"""
    from collections import defaultdict
    
    hash_map = defaultdict(list)
    
    for s in strs:
        # 排序后的字符串作为键
        key = ''.join(sorted(s))
        hash_map[key].append(s)
    
    return list(hash_map.values())

# 示例
strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
print(group_anagrams(strs))
# 输出: [['eat', 'tea', 'ate'], ['tan', 'nat'], ['bat']]
```

### 5.3 最长连续序列

**Leetcode 128. Longest Consecutive Sequence**

```python
def longest_consecutive(nums):
    """使用哈希集合 - O(N)"""
    if not nums:
        return 0
    
    num_set = set(nums)
    max_length = 0
    
    for num in num_set:
        # 只从序列起点开始计数
        if num - 1 not in num_set:
            current = num
            length = 1
            
            while current + 1 in num_set:
                current += 1
                length += 1
            
            max_length = max(max_length, length)
    
    return max_length
```

### 5.4 LRU 缓存

**Leetcode 146. LRU Cache**

```python
class LRUCache:
    def __init__(self, capacity):
        from collections import OrderedDict
        self.cache = OrderedDict()
        self.capacity = capacity
    
    def get(self, key):
        if key not in self.cache:
            return -1
        # 移到末尾（最近使用）
        self.cache.move_to_end(key)
        return self.cache[key]
    
    def put(self, key, value):
        if key in self.cache:
            self.cache.move_to_end(key)
        self.cache[key] = value
        if len(self.cache) > self.capacity:
            # 删除最久未使用的（第一个）
            self.cache.popitem(last=False)
```

## 6. 高频判断

**Q1：哈希表的时间复杂度是多少？**

A：
- 平均情况：O(1) 查找、插入、删除
- 最坏情况：O(N)（所有元素冲突）
- 实际应用中接近 O(1)

**Q2：如何选择哈希冲突解决方法？**

A：
- 链地址法：实现简单，适合冲突较多的情况
- 开放寻址法：缓存友好，适合负载因子较低的情况

**Q3：哈希表 vs 平衡树？**

A：
- 哈希表：平均 O(1)，无序，不支持范围查询
- 平衡树：O(log N)，有序，支持范围查询

**Q4：什么时候用 Set，什么时候用 Map？**

A：
- Set：只需要判断元素是否存在
- Map：需要存储键值对映射关系

**Q5：如何设计一个好的哈希函数？**

A：
- 计算快速
- 分布均匀
- 避免冲突
- 常用方法：除留余数法、乘法哈希、字符串哈希
